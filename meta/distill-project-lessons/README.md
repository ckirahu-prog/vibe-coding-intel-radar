# distill-project-lessons（知识中枢副本）

这是个人全局 skill `distill-project-lessons` 的 GitHub 备份与版本库，用作**跨项目经验中枢**。

Hub 目标仓库：`https://github.com/ckirahu-prog/Skills-and-Experience`

- `SKILL.md` —— 蒸馏工作流（取证 → 分层归因 → 泛用/专用拆分 → 双输出 → 同步 GitHub）。
- `library.md` —— 跨项目泛用经验库（中文，按「人类心得 / AI 执行心得」两大类组织，去重追加，不含任何项目专有名词）。
- `projects/<project-slug>/PROJECT-LESSONS.md` —— 各项目的专有经验文档镜像。

活动副本位于本机 `~/.cursor/skills/distill-project-lessons/`；本目录是其同步备份。
当前项目的专有经验源文件仍在仓库 `docs/PROJECT-LESSONS.md`。

每次复盘后，skill 的 Step 7 会把 `SKILL.md`、`library.md` 和项目经验镜像同步到这里。
